# Basic Chatbot

A simple interactive chatbot written in Python.

## Features
- Responds to greetings such as "hello"
- Answers basic questions like "how are you" and "what is your name"
- Exits when the user types "bye"

## Running the project
1. Open a terminal in this folder.
2. Run:
   ```bash
   python BasicChatbot.py
   ```
3. Type messages at the prompt and enter "bye" to exit.

## Requirements
- Python 3.x
